Two things happened:

1. There's a nice animation

Unfortunately you can't use a simple CSS transition on details elements as they open and close. The workaround is to use JavaScript.

2. Only a single details can be open at a time

If you want to allow multiple, do <accordion-details allow-multiple></accordion-details>

How to use

Include the required script and css files:

<link href="accordion-details.css" rel="stylesheet">
<script src="accordion-details.js"></script>

Now you can use it anywhere, like this:

<accordion-details>
  <details>
    <summary>There is a difference between</summary>
    <p>I am text</p>
  </details>
  <details>
    <summary>knowing the path</summary>
    <p>I am text</p>
  </details>
  <details>
    <summary>and walking the path</summary>
    <p>I am text</p>
  </details>
</accordion-details>

Tips

Use <accordion-details allow-multiple> when you want to, well, allow multiple open sections at once.

To manually open or close, use the platform: <accordion-details open>.

By default, <accordion-details> elements will be styled as a 16:9 flexible box. Images will be scaled to fit as well.
